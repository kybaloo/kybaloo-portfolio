## 🛠 Local Installation

```bash
git clone https://github.com/kybaloo/kybaloo-portfolio.git
cd kybaloo-portfolio
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the portfolio.
